Exercise 5: Creating a Webpage

(Required for Week 8 Assessment)

The professional skills assignment for this week is to use your GitHub account to create a webpage. During the Week 8 Assessment, you will need to direct us to this webpage by providing its URL (it should end in github.io). You will be assessed on the contents of that webpage. To create your page, you should follow the instructions on the GitHub Pages site. This will guide you through the process of creating an appropriately named repository to set up your page. Once you have completed this, you should add to this repository a file named index.html containing the following text: 
<!DOCTYPE html>
<html>

 <head>
 <title>MTH4000 exercise</title>
</head>

 <body>
</body>

</html>
You will be marked on:
  • having a webpage;
  • using the above index.html;
  • entering some text into the body of that webpage (i.e. between <body> and </body>), taking care that nothing else in index.html is altered

    • Note: While you are welcome to later use this webpage for your own use, you should exercise care and only provide information that you are comfortable with. For purpose of the assessment it is not necessary to put any personal information onto that page.
